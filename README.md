# 个性化书籍推荐系统 python1078

## 项目概述

本项目是基于 Python 爬虫技术的个性化书籍推荐系统，采用前后端分离的架构设计。系统主要包括个人中心、书籍管理、公告管理等模块，致力于为用户提供个性化的阅读推荐。

## 技术栈

- **编程语言**: Python
- **开发工具**: PyCharm
- **后端框架**: Flask
- **前端框架**: Vue
- **数据库**: MySQL 5.7+
- **数据库工具**: Navicat、SQLyog

## 功能模块

- 个人中心
  - 用户管理
- 书籍管理
  - 书籍分类管理
  - 热门书籍管理
  - 小说信息管理
- 公告栏管理
- 在线反馈
- 系统管理

## 系统角色

- 用户：浏览、搜索、查看书籍，管理个人资料。
- 管理员：管理用户、书籍、分类、公告等系统内容。

## 运行环境

- Python 环境
- Flask 框架
- Vue 前端框架
- MySQL 数据库

## 部署步骤

1. 准备 Python 运行环境。
2. 安装所需依赖库。
3. 配置 MySQL 数据库。
4. 部署后端 Flask 应用。
5. 部署前端 Vue 应用。
